# Paper To HTML Talk Skill

Build a claim-led narrative: problem pressure, method move, key evidence, limitations, takeaway. Use figure close-reading instead of abstract dumping.

Export with:

```bash
uv run academic-deck html-pptx --deck <deck.yaml> --out <out>
```

Use Beamer only as a formal handout backup.
