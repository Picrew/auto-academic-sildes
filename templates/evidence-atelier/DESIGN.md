# Evidence Atelier

Use for project, profile, or paper decks where source artifacts should feel
selected and installed rather than pasted into a slide.

## Reference DNA

- Huashu: Stamen warmth + Irma Boom catalog tactility.
- JS.Design high-sense pages: image/text asymmetry, restrained accent, curated
  proof surfaces.
- Academic use: one artifact wall, one short claim, one captioned source.

## Grammar

- Warm paper, charcoal text, terracotta accent, sage-green dark reset pages.
- Covers use a small source object; proof pages carry the real visual weight.
- Content pages use irregular bento/index blocks only when each block has a job.
- Evidence images should look like exhibition plates: large, cropped, captioned.
- Avoid glossy cards, mock dashboards, and ornamental gradients.

## Best Fit

- Public researcher profiles with project pages.
- Product-research and systems evidence decks.
- Technical portfolios where screenshots are proof, not decoration.
