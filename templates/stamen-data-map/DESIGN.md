# Stamen Data Map

Purpose: project, benchmark, public-source, and ecosystem decks where evidence
should feel like a mapped terrain.

- Surface: warm cartographic paper, muted green dark pages, ochre signal color.
- Typography: strong sans claims with compact metadata and map-like labels.
- Composition: atlas proof spreads, map-like matrices, bento evidence groups,
  and source object walls.
- Evidence: use large crops and captions to show what relationship or terrain is
  being inspected.
- Cadence: proof pages should feel like plates; content pages should read like
  an index or route map.
- Avoid: decorative maps with no data relationship, stock terrain texture, and
  repeated full-page screenshots.
