# Gallery Proof Room

Use for artifact-rich research profiles and project surveys where the strongest
material is visual: screenshots, figures, paper diagrams, product states, repo
surfaces, or public homepages.

## Grammar

- Warm gallery paper, charcoal ink, and a muted green source accent.
- Oversized proof plates with small wall-label captions.
- Fewer modules per slide; each object should be inspectable.
- Sparse annotations, aligned to the evidence instead of floating over it.
- No nested cards, random glows, stock illustrations, or AI atmosphere.

## Layout

- Cover: `cover-title-wall`
- Evidence: `proof-gallery-split`
- Artifact content: `artifact-showcase`
- Matrix: `matrix-map`
- Labels: `content-bento`

## Image Rules

- Prefer one strong proof plate over several thumbnails.
- Captions should read like exhibition labels: exact object, source, and claim.
- If the proof is a homepage, crop the visible text area or use it only as an
  identity anchor.
- Remove images that do not prove, orient, compare, or show a real artifact.
