# Jetset Theory Grid

Use for conceptual theory, method framing, and bold technical talks that need a
strong typographic reset from conventional academic slides.

## Visual Grammar

- Typography and grid are the primary image.
- Black/cream base with one blue and one red accent.
- Rules, blocks, and table borders are intentionally visible.
- It should feel designed, not decorated.

## Slide Jobs

- Cover: title as poster object, with one source artifact only if useful.
- Content: thesis bullets plus a proof panel; no feature-card mosaics.
- Evidence: screenshot/figure inside a strict bordered proof window.
- Matrix: compressed theory map, not a soft card layout.

## Avoid

- Gradients, shadows, glassmorphism, emoji, and generic AI tech symbols.
- Centered everything.
- Using the style for delicate biographical decks that need warmth.
