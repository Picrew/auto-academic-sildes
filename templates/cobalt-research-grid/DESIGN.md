# Cobalt Research Grid

Use for systems, benchmark, architecture, and tool-use decks where the visual
language should feel technical, exact, and energetic.

## Reference DNA

- Beautiful HTML Templates: Cobalt Grid.
- Swiss technical posters and field manuals.
- Benchmark dashboards only when the table or trace is real evidence.

## Grammar

- Cream paper, hard cobalt grid, black type, minimal decoration.
- Covers can use poster-grid structure; evidence slides use proof ledgers.
- Tables, components, and metrics should align to the grid and read quickly.
- Good default for technical evidence decks; too sharp for delicate biographies.
- Avoid soft shadows, fake UI chrome, and blue-purple AI gradients.

## Best Fit

- Systems and infrastructure talks.
- Benchmarks, leaderboards, agent/tool-use pipelines.
- Technical portfolio decks with real repos or dashboards.
