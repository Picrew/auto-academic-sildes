# Prism Newsroom Index

Use for researcher, lab, or project profiles that have enough public evidence to
behave like a homepage news/index page: selected work, project artifacts, paper
links, talks, and source rows.

## Grammar

- Near-white PRISM surface, slate ink, and one warm source accent.
- Serif title hierarchy with quiet sans metadata.
- Publication/news rows are preferred over generic cards.
- Identity and source crops sit in a left rail; the right side carries the
  public work index.
- No glossy UI, gradient hero, fake dashboard, or decorative profile badges.

## Layout

- Cover: `cover-source-rail`
- Evidence: `proof-dossier`
- Artifact content: `artifact-dossier`
- Matrix: `matrix-ledger`
- Labels: `content-label-board`

## Image Rules

- Use homepage crops as identity anchors, not repeated proof on every slide.
- Project pages, repo surfaces, papers, and benchmark screenshots should carry
  the evidence weight.
- Every image needs a specific caption/source line naming the inspected region.
- If a page feels like a resume card, convert it to a source-linked index row.
