# Forest Editorial Brief

Use for public researcher, paper, or project profiles that need warmth and
editorial maturity without looking like a template.

## Reference DNA

- Beautiful HTML Templates: Editorial Forest and Soft Editorial.
- Academic homepage rhythm: identity first, then selected work, then proof.
- High-sense editorial pages: deep green contrast, paper surface, one dusty
  accent.

## Grammar

- Forest green, warm paper, and a muted rose accent.
- Strongest default for profile decks that need taste and enough density.
- Proof pages use gallery split compositions; source screenshots should be
  cropped and captioned like selected evidence.
- Content pages use bento blocks only when each block has a source or role.
- Avoid gradients, generic cards, stock portraits, and decorative screenshots.

## Best Fit

- Researcher and lab profiles.
- Paper/project intros with public source artifacts.
- Academic talks that need to feel polished but not theatrical.
