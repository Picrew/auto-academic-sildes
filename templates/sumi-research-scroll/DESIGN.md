# Sumi Research Scroll

Use for academic profiles, paper dossiers, and source-linked reviews that should
feel like a serious research homepage translated into an exhibition handout.

## Grammar

- Warm paper, near-black ink, and a restrained red source cue.
- Serif headline rhythm with sans metadata; proof and publication rows behave
  like a scroll/index rather than app cards.
- Vertical rules, single-column metric ledgers, and marginal evidence notes
  create density without glossy decoration.
- Borrow academic-homepage information architecture, not the surface of any one
  website.

## Layout

- Cover: `cover-source-rail`
- Evidence: `proof-marginalia`
- Artifact content: `artifact-marginalia`
- Matrix: `matrix-ledger`
- Labels: `content-marginalia`

## Image Rules

- Identity images belong in the cover/source rail; proof slides need papers,
  repos, benchmark surfaces, project pages, or tightly cropped homepage text.
- Captions should read like source notes, not marketing copy.
- Use full-frame crops only when the file was already prepared as a readable
  source plate.
- Do not use repeated homepage screenshots as a substitute for work evidence.
