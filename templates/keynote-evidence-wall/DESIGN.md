# Keynote Evidence Wall

For demos, product-research talks, and visual technical stories.

## Feel

Polished and memorable, but still technical. It should show real artifacts rather
than brand theater.

## Layout

- big claim
- large proof image or before/after wall
- minimal bullets
- one metric strip
- optional product/demo flow

## Visual Choices

- strong image scale
- high contrast
- sparse text
- 1-2 visual accents
- crisp screenshot crops

## Avoid

- stock imagery
- glossy UI cards inside cards
- unsupported "wow" claims
- tiny captions that only decorate
