# Huashu Editorial Lab

Use for academic/technical talks that need Huashu-style typographic confidence:
hard grid, strong hierarchy, one accent color, and no generic AI technology
cliches.

## Grammar

- Black/ivory base with one red-orange signal color.
- Helvetica-like heavy titles, ledger rows, and strict alignment.
- The grid is the visual system; decoration is only allowed when it encodes
  content.
- High contrast is used for hierarchy, not spectacle.
- Avoid purple-blue gradients, cyber neon, glowing dashboards, and random blobs.

## Layout

- Cover: `cover-poster-grid`
- Evidence: `proof-ledger`
- Artifact content: `artifact-ledger`
- Matrix: `matrix-ledger`
- Labels: `content-field-manual`
- Metrics: `metrics-ledger`

## Image Rules

- Proof images sit inside hard frames and must occupy enough real area to be
  readable in a room.
- Callout pins must land on source pixels, not letterbox whitespace.
- If a caption/source line drops below readable contrast, the grammar fails.
