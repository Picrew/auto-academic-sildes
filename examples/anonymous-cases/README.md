# Anonymous Cases

Three synthetic fixtures for testing agent-driven deck construction without exposing real names, institutions, private screenshots, or harvested profile material.

- `case-a-paper/`: paper-talk style routing study.
- `case-b-system/`: system launch-gate review.
- `case-c-workspace/`: anonymous workspace/profile index.

The evidence images in `assets/images/` are generated local fixtures. Rendered contact sheets are copied to `docs/anonymous_case_gallery/` for quick visual review.
